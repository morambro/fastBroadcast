\section{Implementation}
\label{sec:implementation}

Our application is structured in different units, each performing a specific function. There is a single Activity class\footnote{Entry point of our Android application. For other information, visit \url{http://developer.android.com/guide/components/activities.html}}, called \ttt{FastBroadcastActivity}, which at application startup performs graphical initializations, and creates the application controller.
